# tsuki-no-soko-adv-mvp

推理ADV『月の底で、息をする』Ren'Py 8系MVPプロトタイプ。

2056年の月面南極都市《白環市》を舞台に、証拠品入手・聞き込み・最終推理・エンディング分岐までを仮素材で確認するための最小実装です。

## 推奨ファイル構成

```text
ren-py-8-adv-mvp-2/
  game/
    options.rpy      # 画面サイズ・タイトルなどの基本設定
    characters.rpy   # キャラクター定義と仮背景
    evidence.rpy     # 証拠品データ、章進行、推理問題、add_evidence()
    screens.rpy      # 証拠一覧・証拠提示スクリーン
    script.rpy       # 本編進行、聞き込み、最終推理、エンディング分岐
```

## 実装済みの骨組み

- 証拠品IDベースのカタログ管理
- `add_evidence(id)` による証拠入手
- 章番号・章タイトルの進行管理
- 証拠品一覧スクリーン
- 聞き込み対象選択メニュー
- 証拠提示式の最終推理
- 正解・不正解・ヒント表示
- True / Normal / Bad Ending 分岐
