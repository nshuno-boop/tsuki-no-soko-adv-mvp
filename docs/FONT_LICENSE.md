# Font License Notes

The project uses Noto Sans CJK Japanese fonts for Japanese text rendering in Ren'Py.

## Bundled Files

- `game/fonts/NotoSansJP-Regular.otf`
- `game/fonts/NotoSansJP-Bold.otf`

## Source

Noto Sans CJK is maintained by the Noto Fonts project:

https://github.com/notofonts/noto-cjk

The bundled files correspond to the Japanese OTF fonts:

- `Sans/OTF/Japanese/NotoSansCJKjp-Regular.otf`
- `Sans/OTF/Japanese/NotoSansCJKjp-Bold.otf`

## License
