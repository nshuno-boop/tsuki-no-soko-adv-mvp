# ALPHA GOALS

## Phase 5 / Alpha Polish

Alpha版では、仮素材のままでも「最初から最後まで遊び、事件の構造と感情の着地を確認できる」状態を目標にする。

## Alphaで達成したい状態

- プロローグからTrue / Normal / Bad Endingまで通しプレイできる
- 日本語フォントが正常に表示される
- 証拠品、人物メモ、タイムライン、最終推理が実機で表示される
- 各章の目的と調査まとめで、プレイヤーが迷いすぎない
- 最終推理の問題文、ヒント、正解・不正解メッセージが自然に読める
- True Endingで、徹の本心とノアの感情が最低限回収される

## Betaでやること

- シナリオ全体の文量とテンポを調整する
- 聞き込みの追加会話を増やす
- BGM/SEを仮導入し、場面転換の手触りを整える
- タイトル画面、セーブ画面、環境設定画面など標準UIを作品トーンに寄せる
- 最終推理のリトライ体験をさらに分かりやすくする

## 正式版でやること

- 背景・立ち絵・UI素材を本素材へ差し替える
- 演出、画面効果、音響の完成度を上げる
- ルート分岐や追加エピソードを検討する
- 誤字脱字、表記ゆれ、推理難度をプレイテストで調整する
